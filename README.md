Guía de uso archivo CronometroTransferenciasDeArchivos.ps1
------------------------------------------------------------

Cambiar $carpetaDestino por la ruta donde se recibirá el archivo, si se usa LocalSend se puede cambiar la misma en el menú de configuraciones de la aplicación.
Cambiar $nombreArchivoEsperado por el nombre del archivo que se recibirá junto con su extensión (.exe, .png, .pdf, .iso, etc.)
Poner en logfile la ruta del archivo .txt donde se guardarán los registros de tiempo. 
Ejecutar el archivo en Windows Powershell antes de comenzar la transferencia del archivo deseado.
